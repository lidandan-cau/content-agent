## 🏠 家庭管理类（6条）

**① 冰箱食材变菜谱**
```
冰箱里有：【西红柿、鸡蛋、豆腐、猪肉、青椒】
家里有【3】口人，孩子【9】岁，不吃【香菜】
请生成今晚的晚餐菜单，2-3道菜，30分钟内能做完，附上步骤。
```

**② 采购清单整理**
```
我下周要做以下菜：【西红柿炒蛋、红烧排骨、炒青菜、番茄豆腐汤】
家里已有：【盐、酱油、鸡蛋、葱姜蒜】
请帮我列出需要采购的食材清单，按生鲜/调料/其他分类，并估算大概费用。
```

**③ 家务清单自动分配**
```
家庭成员：我（上班族，晚7点到家）、老公（晚8点到家）、孩子9岁
本周家务：买菜、做饭、洗碗、拖地、洗衣晾衣、辅导作业、倒垃圾、擦桌子
请帮我制定一份本周家务分工表，要求相对公平，孩子承担力所能及的部分。
```

**④ 孩子日程规划**
```
孩子【9】岁，本周课外安排：【周二钢琴课5-6点，周四游泳课4-5点，周六画画课上午10-12点】
请帮我规划孩子本周每天放学后到睡前（下午4点-9点）的时间安排，
包括：作业时间、休息时间、自由玩耍时间，
要求：不要太满，给孩子留出自由时间，睡前有安静过渡时间。
```

**⑤ 快速整理思路**
```
我脑子里现在有很多乱的事情需要处理：
【1. 孩子下周要交手工作业
2. 婆婆下周来住，需要准备房间
3. 工作上有个方案周四要汇报
4. 自己想开始写公众号但不知道从哪里开始
5. 家里洗衣机好像有问题需要维修】

请帮我：按紧急程度排序，列出今天/明天/这周分别需要做什么。
```

**⑥ 育儿问题快速查询**
```
孩子【9】岁，今天出现了以下情况：【发烧38.2度，精神还好，没有其他症状】
请告诉我：
1. 现在应该怎么处理（物理降温/吃药/就医时机）
2. 哪些情况出现了必须立即去医院
3. 接下来24小时需要注意什么
注意：我需要的是参考信息，最终以医生建议为准。
```

## 👩 个人成长类（5条）

**⑦ 情绪复盘**
```
今天/这周让我情绪波动的事：
【早上催孩子上学又发脾气了，说了"你怎么这么不让人省心"这样的话】

请帮我分析：
1. 这件事里，是哪条我的隐性需求/边界被触碰了？
2. 我说的那句话，孩子听到后会有什么感受？
3. 下次遇到同样情况，我可以怎么说？（给2-3个替代方案）
不需要安慰我，帮我看清楚就好。
```

**⑧ 读书笔记整理**
```
我最近读了《【被讨好的人生】》，以下是我觉得重要的摘录和自己的想法：
【粘贴你的摘录或随手记的想法】

请帮我：
1. 整理成结构清晰的读书笔记（核心观点+我的感悟）
2. 找出其中与我个人经历最相关的1-2个点，展开说说
3. 生成3个我可以立刻实践的行动
```

**⑨ 写作开头破题**
```
我想写一篇关于【心理边界】的文章，主题是【为什么内疚感不是爱】
目标读者：30-45岁的妈妈，有过对孩子过度付出然后感到疲惫的经历
请给我5个不同风格的开头（故事型、问句型、金句型、场景型、反差型），
每个开头50字以内，让我选一个方向继续写。
```

**⑩ 目标拆解**
```
我有一个目标：【三个月后开始有稳定的写作收入，每月1000元以上】
现在的情况：【有写作习惯但没有发布过文章，不知道从哪个平台开始，每天有1小时空闲时间】

请帮我：
1. 判断这个目标是否现实，如果不现实说明原因
2. 如果可行，拆解成每周具体行动（第1-4周分别做什么）
3. 告诉我最容易失败的环节是哪里，以及预防方案
```

**⑪ 自我介绍生成**
```
我需要写一段自我介绍，用于【公众号简介/社群自我介绍/朋友圈置顶】
我的情况：39岁，顺义妈妈，孩子9岁，正在经营个人品牌，
核心内容方向：心理边界 + AI效率工具 + 山水疗愈
我的风格：温暖真诚，不端着，像朋友聊天

请写3个版本：
- 简短版（30字以内）
- 标准版（80字以内）
- 详细版（150字以内）
```

## 💼 职场与沟通类（5条）

**⑫ 邮件/消息润色**
```
请帮我润色以下这段话，让它更专业/更有礼貌，但保留我的意思：
场景：【发给领导/客户/老师/家长群】
原文：【粘贴你想发的话】
要求：不要太正式，保持真诚；如果原文有不妥的地方请指出。
```

**⑬ 拒绝话术**
```
我需要拒绝以下请求，但不想让对方太尴尬，也不想撒谎：
情况：【闺蜜邀请我周末聚会，但我真的不想去，想在家休息】
关系：【好朋友，但我最近需要独处充电】

请给我2-3种不同方式的拒绝话术：
- 温和型（给对方台阶下）
- 直接型（真实说明原因）
- 缓冲型（改变时间/方式而不是直接拒绝）
```

**⑭ 汇报/发言稿**
```
我需要做一个【5分钟】的汇报/发言，主题是：【这个月的工作进展】
关键信息：
- 完成了什么：【推进了XX项目，完成了XX报告】
- 遇到了什么困难：【XX问题还在解决中】
- 下一步计划：【下周完成XX，月底交付XX】
- 我想给领导留下的印象：【有条理、执行力强】

请帮我组织成清晰的发言稿，口语化，有逻辑，不啰嗦。
```

**⑮ 朋友圈/小红书文案**
```
我想发一条【朋友圈/小红书】，记录以下这件事：
【今天带孩子去怀柔山里玩，傍晚看到漫山红霞，孩子说"妈妈这个颜色叫什么"，
我说"叫晚霞"，他说"我觉得应该叫妈妈色"】

请帮我写：
1. 适合朋友圈的版本（100字以内，有温度，不矫情）
2. 适合小红书的版本（有标题，200字，附3-5个话题标签）
```

## 🌿 情绪与自我照顾类（4条）

**⑯ 崩溃时的快速梳理**
```
我现在情绪很差，需要你帮我理一理，不要安慰我，帮我想清楚。
发生了什么：【描述让你崩溃的事】
我现在的感受：【愤怒/委屈/疲惫/无助，选一个或多个】
我最担心的是：【说出你内心最深的恐惧或担忧】

请帮我：
1. 用一句话概括这件事的核心矛盾
2. 这件事里哪些是我能控制的，哪些不是
3. 接下来24小时，我最重要的一件事是什么
```

**⑰ 和孩子道歉**
```
我刚才对孩子发脾气，说了一些过激的话，现在想去道歉和修复关系。
发生了什么：【我因为孩子写作业拖延，忍不住发火，说了"你真让我失望"】
孩子年龄：【9岁】

请帮我：
1. 告诉我怎么开口，第一句话说什么
2. 完整的道歉和修复对话（我说什么，如果孩子这样回应我再说什么）
3. 这次之后，如何避免下次同样的爆发
```

**⑱ 独处计划设计**
```
我需要给自己安排一次独处的时间，好好充电，大概【半天/一天】
我在【北京顺义】，不想走太远，喜欢安静的自然环境，不喜欢人太多的景区
预算：【200元以内】，有车

请帮我规划一个真实可执行的独处半天/一天方案，包括：
去哪里、几点出发、带什么、大概怎么安排时间
不需要排得很满，主要是让我能真正放空。
```

**⑲ 写给自己的信**
```
请帮我写一封信，写信人是"五年后更好的自己"，收信人是"现在的我"。
现在的我的情况：【39岁，刚离职，正在探索新方向，有点焦虑但又有点期待】
我最需要听到的话：【不是鼓励，而是真实的、有点温柔有点直接的话】
信的长度：300字左右，不要太煽情，要真诚。
```

**⑳ 睡前放松引导**
```
我今晚思绪很乱，睡不着，请给我一段睡前放松引导词。
我目前的状态：【脑子里一直在想明天的事，身体很紧】
时长：3-5分钟能读完
风格：温柔平静，不要太"课程感"，像一个朋友在陪着我
最后用一句话结尾，帮我把注意力带回当下。
```

## 💡 使用小贴士

**复制后只需改【】里的内容**，其他部分保持不变效果最好。

**结果不满意时可以说：**
- "太正式了，更像朋友说话的感觉"
- "第二点再展开一些"
- "换一个角度重新试试"
- "用更温暖的语气重写"
