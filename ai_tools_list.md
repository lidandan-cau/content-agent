## 🍳 场景一：一周菜谱生成

**推荐工具：** ChatGPT / Kimi / 文心一言

**用法：** 每周一次，5分钟搞定全周晚餐，告别"今天吃什么"内耗。

**详细提示词：**
```
我家有【3】口人，孩子【9】岁。
不吃的食材：【香菜、芹菜、海鲜】
口味偏好：【清淡为主，偶尔川味】
请帮我生成这周5天（周一到周五）的晚餐菜谱，要求：
1. 每天2-3道菜，30分钟内能做完
2. 食材尽量共用，减少采购种类
3. 营养均衡，有荤有素
4. 最后附上本周需要采购的食材清单
```

## 📝 场景二：家长会发言稿

**推荐工具：** ChatGPT / Claude / Kimi

**用法：** 家长会前一天，15分钟生成一篇真诚有温度的发言稿。

**详细提示词：**
```
请帮我写一篇家长会发言稿，要求口语化、真诚，不要说教。
孩子情况：
- 年级：【小学三年级】
- 这学期的进步：【数学成绩提升了，开始主动预习】
- 还需要改进的地方：【写作业拖延，需要大人提醒】
- 孩子的一个可爱细节：【最近迷上了画机器人，会画给同学看】

发言时长：【3分钟】
语气要求：温暖、真实，像在和老师聊天，不要用"首先其次最后"这种套路结构。
```

## 🧘 场景三：情绪复盘与分析

**推荐工具：** Claude / ChatGPT

**用法：** 每周五晚上，把这周让你难受的事倒给AI，找出情绪规律。

**详细提示词：**
```
我想做一次情绪复盘，请帮我分析。
以下是我这周让我情绪波动的几件事：
【1. 周一早上催孩子上学，我发火了，说了很重的话
2. 周三开会时被领导当众否定，心里很委屈
3. 周五和老公因为家务分配争吵了】

请帮我：
1. 找出每件事背后，我的哪条"边界"被触碰了
2. 分析我的情绪触发规律（什么场景最容易让我崩溃）
3. 给出下次遇到类似情况时，我可以说的1-2句话
语气建议：像一个温柔但直接的朋友，不要安慰我，帮我看清楚就好。
```

## ✍️ 场景四：文章润色与改稿

**推荐工具：** Claude / Kimi

**用法：** 写完初稿后，让AI帮你提升文字质量，保留你的语气。

**详细提示词：**
```
请帮我润色以下这段文字，要求：
1. 保留我原有的口语化风格和"我"的视角，不要变成正式文章
2. 让句子更有节奏感，读起来更流畅
3. 找出其中逻辑不清晰或表达重复的地方，告诉我并修改
4. 如果有一句话可以升华主题，帮我加在结尾

我的原文：
【粘贴你的文字】
```

## 🏠 场景五：家务分配方案

**推荐工具：** ChatGPT / Kimi

**用法：** 把家庭成员情况告诉AI，让它给出公平的分配方案，减少家庭争议。

**详细提示词：**
```
请帮我制定一份家务分配方案，要求尽量公平，减少争执。
家庭成员：
- 我：【全职/上班族，每天几点到家】，擅长：【做饭、整理】
- 老公：【上班族，每天几点到家】，擅长：【修理、采购】
- 孩子：【9岁】，可以承担简单任务

家务清单：做饭、洗碗、拖地、采购、洗衣、辅导作业、倒垃圾、整理客厅

请按照每日/每周频率，给出分工建议，并说明这样分配的理由。
如果某一方明显更累，请指出来并提出调整建议。
```

**推荐工具：** ChatGPT / 讯飞星火

2. 分析每个知识点的错误原因（是概念不清还是粗心）
3. 针对每个薄弱点，给出2-3道同类型练习题
4. 制定一个两周的复习计划，每天不超过20分钟

错题内容：
【粘贴错题或描述错误情况】
```

## 💬 场景七：亲子沟通话术

**推荐工具：** Claude / ChatGPT

**用法：** 遇到和孩子沟通困难的场景，让AI帮你换一种说法。

**详细提示词：**
```
我和孩子之间出现了以下情况，请帮我换一种更好的沟通方式：

情况描述：【孩子每天写作业要拖两个小时，催他就发脾气，不催就一直玩】

我现在通常说的话：【"你怎么这么慢！别人都写完了！"】

请帮我：
1. 分析我现在的说法为什么没效果
2. 给出3种不同语气的替代说法（温和型、直接型、引导型）
3. 说明每种说法背后的沟通逻辑
要求：基于"课题分离"原则，让孩子感受到被尊重而不是被指责。
```

## 🗓️ 场景十：周计划制定

**推荐工具：** ChatGPT / Claude / Notion AI

**用法：** 每周日晚上，花10分钟和AI一起规划下周，把重要的事放进去。

**详细提示词：**
```
请帮我制定下周的个人计划，我的情况如下：

想推进但容易被拖延的事：【每天写500字、每周运动2次、读完手边的书】
我的可用时间：【工作日早上7-8点、午休1小时、晚上9点后1小时；周末相对自由】
我容易在哪里失败：【晚上刷手机超时、周末计划太满反而什么都没做】

1. 把任务合理分配到每一天
2. 留出缓冲时间，不要排满

## 💪 场景十二：运动计划制定

**推荐工具：** ChatGPT / Kimi

**用法：** 根据你的实际情况，生成一个真的能坚持的运动计划。

**详细提示词：**
```
请帮我制定一个适合我的运动计划，要求简单、真实可执行，不要过于理想化。

我的情况：
- 体能基础：【较弱，久坐，偶尔散步】
- 目标：【减脂5kg / 增强体力 / 缓解腰背酸痛】
- 可用时间：【工作日早上20分钟，周末1小时】
- 场地限制：【只有家里，没有健身房】
- 以往失败原因：【计划太难坚持不下去、下雨天就断了】

请给出：
1. 第一个月的具体计划（每天做什么，几组几次）
2. 如果某天没时间，5分钟最简版是什么
3. 如何判断计划是否需要调整
```

## 💡 用好AI的三个关键

**1. 背景越具体，答案越有用**
"帮我写发言稿"得到的是模板；"帮我写一篇3分钟的家长会发言，孩子三年级，这学期数学进步了，但写作业拖延"得到的才是真正能用的内容。

**2. 不满意就继续说**
第一次结果不好，直接说："太正式了，更口语化一点" / "第二点展开说说" / "换一个角度再试一次"。AI不会不高兴。

**3. 把有效的提示词存下来**
遇到一个好用的提问方式，存进备忘录。下次同类问题直接调用，越用越顺手。
